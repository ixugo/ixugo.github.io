循环

while 语句

循环三要素：起点，终点，数据变化规律

for 与 while 的区别：从 structure 来说，for 比较严谨，while 比较自由

break or continue

break：直接跳出循环，任意位置执行即直接跳出循环，无视后面的语句

continue：结束当次循环

1
2
3
4
5
6
7
8
9

// 求 1-10 奇数

int sum=0;
for (int i=1; i<=10; i++)
{
  if(i%2) continue;     // 如果是偶数就跳出本次循环
  sum+=i;          		  // 否则就相加求和
}
printf("%d",sum);
